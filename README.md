> project: FERNS AND PETALS
> type: Data Analytics | Excel Dashboard
> status: ✅ Completed
> stack: Excel | Pivot Tables | Power Query | Dashboard Design
📌 Overview
End-to-end sales analysis project for a gift & occasion-based retail business — from raw CSVs to a fully interactive dashboard.
📂 Data Sources
├── customers.csv     → 100 records
├── orders.csv         → 1000 records
├── products.csv       → 70 records
🔧 Build Log
[✔] Imported & cleaned raw data (Customers, Orders, Products)
[✔] Structured relational data across sheets
[✔] Built Pivot Tables → Revenue by Month, Category & Occasion
[✔] Calculated Avg. Order-to-Delivery time gap
[✔] Ranked top products by Revenue & Quantity sold
[✔] Designed interactive Dashboard with Slicers
[✔] Validated totals → ₹35,20,984 total revenue tracked
📊 Key Insights
→ Top categories   : Soft Toys, Sweets, Colors, Mugs
→ Peak occasions    : Diwali, Raksha Bandhan, Anniversary
→ Revenue seasonality tracked across 12 months
→ Avg fulfillment gap: ~5.5 days

Excel Pivot Tables Pivot Charts Data Cleaning Dashboard Design Business Reporting

Turned messy transactional data into a clean, decision-ready dashboard — the same workflow real Data Analysts use daily to drive business decisions.
